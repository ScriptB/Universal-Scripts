# Tools

This folder contains utility scripts and testing tools.

## Files

### UNC_Compatibility_Test.lua
Tests executor compatibility with UNC (Universal Naming Convention) functions:
- API availability testing
- Feature support detection
- Comprehensive compatibility report
- Safe error handling

### UNC_Security_Check.lua
Tests executor security and vulnerability mitigation:
- Dangerous API access testing
- Security vulnerability assessment
- Robux protection verification
- Bypass detection

### DevCopy.lua
Developer console copier:
- Captures console output
- Automatic logging
- Debug assistance
- Error tracking

### KeySystem.lua
Authentication and key validation system:
- Secure key verification
- User authentication
- Access control
- License management
