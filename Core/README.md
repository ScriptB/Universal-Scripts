# Core Scripts

This folder contains the main scripts that form the foundation of the ScriptB Universal Scripts collection.

## Files

### Universal_Aimbot.lua
The main aimbot script with comprehensive features:
- Advanced aimbot functionality
- ESP features
- Movement enhancements
- Performance monitoring
- Professional UI using BracketLib

### Loader.lua
Centralized loader script that handles:
- Key system validation
- Script loading order
- Error handling
- Dependency management

## Usage

### Direct Load
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/ScriptB/Universal-Scripts/refs/heads/main/Core/Universal_Aimbot.lua"))()
```

### Loader (Recommended)
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/ScriptB/Universal-Scripts/refs/heads/main/Core/Loader.lua"))()
```

## Dependencies

- **BracketLib**: UI library (loaded automatically)
- **DevCopy**: Console copier (optional)
- **KeySystem**: Authentication system (if using loader)
